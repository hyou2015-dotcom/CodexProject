# CodexProject - Tetris

GitHub Pages 向けのシンプルなテトリス実装です。

## 遊び方
- `← / →`: 左右移動
- `↑`: 回転
- `↓`: ソフトドロップ
- `Space`: ハードドロップ
- `P`: 一時停止
- 画面右側の「落下速度」スライダー: ゲーム速度を `0.5x`〜`2.0x` に変更
- スマホ: 画面下の操作ボタン（← / ⟳ / → / ↓ / DROP / P）で操作

## GitHub Pages で公開する
このリポジトリは `main` ブランチの `/ (root)` を Pages ソースに設定すれば、
`index.html` がそのまま公開されます。

公開 URL 例:
`https://hyou2015-dotcom.github.io/CodexProject/`
